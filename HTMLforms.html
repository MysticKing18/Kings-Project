HTML < input > TYPES

1. < input type="button" > - Creates a clickable button.



2. < input type="checkbox" > - Creates a checkbox for multiple selections.





3. < input type="color" > - Lets the user pick a color.



4. < input type="date" > - Lets the user select a date.



5. < input type="datetime-local" > - Lets the user select date and time.



6. < input type="email" > - For entering an email address.



7. < input type="file" > - Lets the user upload a file.



8. < input type="hidden" > - Holds hidden data (not visible).


9. < input type="image" > - Image used as a submit button.



10. < input type="month" > - Lets user select month and year.



11. < input type="number" > - For entering numbers.



12. < input type="password" > - For entering passwords (hidden text).



13. < input type="radio" > - For selecting one option only.



14. < input type="range" > - Slider to select a value.



15. < input type="reset" > - Resets the form fields.



16. < input type="search" > - Search input field.



17. < input type="submit" > - Submits the form.



18. < input type="tel" > - For entering telephone numbers.



19. < input type="text" > - Simple text input.



20. < input type="time" > - For selecting a time.



21. < input type="url" > - For entering website URLs.



22. < input type="week" > - Select week and year.
